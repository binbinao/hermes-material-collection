## 文章概览

这篇文章系统介绍了 **Hermes Agent**（一个可自托管的 AI Agent 平台）的 10 个典型应用场景，强调其"持续在线、带记忆、多步骤工作流自动化、支持子 Agent 团队协作"等核心能力。

## 主要章节

### 引言
Hermes Agent 是一个具备持久记忆和多步骤自动化能力的 always-on 助手，支持子 Agent 团队并行执行任务。

### 10 个使用场景

1. **持久化 Agent**：运行能在长期运行中学习工作流的常驻 Agent。
2. **内容创作流水线**：将研究 → 大纲 → 撰稿 → 发布等阶段自动化串联。
3. **并行子 Agent**：把大任务拆给多个子 Agent 并行处理。
4. **自然语言系统操作**：用普通英语指令完成系统级任务。
5. **外部服务编排**：把第三方服务接入自动化流水线。
6. **部署流水线**：自动化 CI/CD 并能处理边界情况。
7. **定时简报**：调度自动化的日常 briefing。
8. **自动化研究与数据处理**：收集与加工数据。
9. **长任务跨重启续跑**：服务器重启后状态不丢，工作流继续。
10. **全栈组合**：把上述所有能力组合到一个完整例程中。
