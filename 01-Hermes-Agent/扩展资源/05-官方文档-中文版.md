> 来源：https://hermes-agent.org/zh

---

## 产品定位

Hermes Agent 是由 Nous Research 开发的开源自主 AI 智能体。

- **不是**代码补全工具
- **不是**套壳聊天机器人
- 是运行在用户服务器上的**持久个人智能体**
- 可连接消息账号，学习用户项目、自动构建技能
- 支持多平台触达

**核心特点**：持久记忆、自动技能创建、多平台接入整合在同一个开源包中。

---

## 核心功能

### 1. 持久记忆

跨会话记录用户的偏好、项目信息和环境配置，使用时间越长对用户的理解越深入，无需每次重新解释上下文。

### 2. 自动技能创建

解决复杂问题后会自动生成可复用的技能文档，技能永久保存，支持搜索和分享，兼容 `agentskills.io` 开放标准。

### 3. 多平台消息网关

通过单一网关进程可连接 Telegram、Discord、Slack、WhatsApp、Signal 和 CLI，支持语音备忘录转录，跨平台对话无缝切换（例如在 Telegram 开启对话，可在终端继续）。

### 4. 定时自动化任务

内置 cron 调度器，支持向任意平台推送通知，可配置每日报告、夜间备份、每周审计、晨间简报等无人值守任务。

### 5. 并行子智能体

可生成隔离的子智能体处理并行工作流，每个子智能体拥有独立的对话和终端，通过 RPC 实现多步骤流水线处理，零上下文消耗。

### 6. 完整浏览器与网页控制

支持网页搜索、页面内容提取、完整浏览器自动化（导航、点击、输入、截图），同时支持视觉分析、图像生成、文字转语音和多模型协作推理。

## MLOps 与 AI 训练专属能力

Hermes Agent 不仅是任务自动化工具，还是生成训练数据、运行 RL 实验、导出轨迹用于微调的完整平台：

1. **批量处理**：可并行生成数千条工具调用轨迹，支持自动检查点，可配置工作进程数、批次大小和工具集分布。
2. **强化学习训练**：集成 Atropos 框架，支持对智能体行为进行强化学习，内置 11 种工具调用解析器，支持任意模型架构训练。
3. **轨迹导出**：支持以 ShareGPT 格式导出对话用于模型微调，自带轨迹压缩功能，可将训练数据压缩至 token 预算范围内。

## 快速使用指南

支持 Linux、macOS 和 WSL2，无需任何前置依赖，一键自动安装所有组件。

### 1. 安装

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

自动完成 uv、Python 3.11 安装，克隆仓库并配置所有环境，无需 sudo 权限。

> ⚠️ 原生 Windows 支持仍处于实验阶段，需安装 WSL2 后在其中运行 Hermes Agent。

### 2. 配置

```bash
hermes setup    # 启动交互式设置向导
hermes model    # 选择使用的大模型
```

支持连接 Nous Portal（OAuth）、OpenRouter（API Key）或自定义端点。

### 3. 开始对话

```bash
hermes          # 启动完整交互式 CLI
```

### 4. 接入多平台（可选）

```bash
hermes gateway setup    # 启动交互式网关设置向导
hermes gateway         # 启动消息网关
hermes gateway install # 将网关安装为系统服务
```

### 5. 保持更新

```bash
hermes update    # 拉取最新代码并重新安装依赖
```
