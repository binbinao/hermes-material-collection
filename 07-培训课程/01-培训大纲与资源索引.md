## 一、培训目标

完成本培训后，学员将能够：
1. 理解 Hermes 技术生态全貌（Agent、LLM、RAG、Edge Inference、Math Reasoning）
2. 部署和配置 Hermes Agent，掌握核心功能（记忆、技能、网关）
3. 评估 Hermes 4 模型在各场景下的适用性
4. 对比 Hermes Agent 与竞品（OpenClaw 等），做出合理选型
5. 了解 Hermes 在学术前沿的应用（RAG 优化、数学推理）

## 三、实操练习

### Lab 1: Hermes Agent 快速部署 (15min)

```bash
# 安装
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

# 交互式配置
hermes setup

# 首次对话
hermes "介绍一下你自己"
```

### Lab 2: 记忆系统体验 (20min)

```bash
# 启动 Hermes
hermes

# 告诉 Hermes 一些偏好
"记住：我偏好中文交流，变量名保留英文"

# 退出后重新进入，验证记忆
hermes "你还记得我的偏好吗？"

# 查看记忆文件
cat ~/.hermes/memories/MEMORY.md
cat ~/.hermes/memories/USER.md
```

### Lab 3: 技能系统体验 (20min)

```bash
# 让 Hermes 完成一个复杂任务
"帮我写一个 Python 脚本，监控指定目录的文件变化"

# 查看自动生成的技能
ls ~/.hermes/skills/

# 再次请求类似任务，观察技能复用
"帮我写一个日志文件监控脚本"
```

### Lab 4: 多平台网关配置 (25min)

```bash
# 交互式网关配置
hermes gateway setup

# 启动网关
hermes gateway

# 安装为系统服务
hermes gateway install
```

## 五、企业落地指南

### RDSHermes（企业版）

开源版 Hermes 是开发者工具，RDSHermes 是团队产品：
- 零配置云控制台启动
- 内置 WebUI（非技术用户可用）
- 加密凭证管理
- 预装专业技能（数据库诊断、慢查询分析、索引优化）
- 完整审计追踪
- 多数据库支持（MySQL、PostgreSQL、SQL Server、MariaDB）

### 团队部署建议

1. **Week 1**: 单人试用 Hermes Agent，积累 2-3 个核心技能
2. **Week 2**: 部署 Gateway，接入团队常用 IM（飞书/钉钉/企微）
3. **Week 3**: 技能治理 — 清理冗余技能，建立技能命名规范
4. **Week 4**: 评估是否需要 RDSHermes 企业版
